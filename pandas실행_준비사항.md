VS Code가 이미 설치되어 있다면 절반은 끝난 셈입니다! 이제 VS Code 안에서 파이썬 데이터 분석을 할 수 있도록 **'엔진(Python)'**과 **'확장 도구(Extension)'**를 연결해 줄 차례예요.
다음 단계를 차근차근 따라와 주세요.


1단계: 파이썬(Python) 엔진 설치
VS Code는 일종의 메모장 같은 '편집기'일 뿐이라, 실제로 코드를 계산할 파이썬 본체가 필요합니다.
1. python.org에 접속해서 최신 버전을 다운로드하세요.
1. 중요: 설치 창이 뜨면 하단의 [Add Python to PATH] 체크박스를 반드시 체크하고 'Install Now'를 누르세요. (이걸 안 하면 나중에 명령어가 안 먹혀요!)


2단계: VS Code에서 파이썬 확장팩 설치
VS Code가 파이썬 언어를 알아먹게 만들어야 합니다.
1. VS Code 왼쪽 메뉴 바에서 네모 네 개가 그려진 아이콘(Extensions, 단축키 Ctrl+Shift+X)을 클릭하세요.
1. 검색창에 **Python**을 입력하고, Microsoft에서 만든 파이썬 확장팩을 [Install] 하세요.
2. 	가장 위에 있는 Python (Microsoft 제작)
• 		파란색 체크 표시가 있고 다운로드 수가 210M인 바로 그겁니다. 이것 하나면 디버깅, 코드 자동 완		성 등 필수 기능을 다 갖추게 됩니다.
• 		두 번째에 있는 Python Debugger (Microsoft 제작)
• 		최신 VS Code 버전에서는 파이썬 실행과 디버깅을 위해 이 확장팩도 세트로 설치하는 것을 권장
1. 추가로 **Jupyter**도 검색해서 설치해 주세요. (데이터 분석의 필수템인 주피터 노트북을 VS Code 안에서 쓰게 해줍니다.)
2. 	'데이터 분석 실습'을 할 때, 엑셀처럼 코드를 칸마다 실행할 수 있게 해주는 핵심 도구


3단계: 분석용 필수 라이브러리 설치 (터미널 이용)
커리큘럼 1일차에 나온 라이브러리들을 내 컴퓨터에 깔아주는 과정입니다.
1. VS Code 상단 메뉴에서 **[Terminal] -> [New Terminal]**을 엽니다 (화면 하단에 검은 창이 뜹니다).
1. 아래 명령어를 복사해서 붙여넣고 엔터를 치세요:
Bash
pip install pandas numpy matplotlib beautifulsoup4 notebook

• pip는 파이썬의 앱스토어 같은 거예요. 필요한 도구들을 한꺼번에 설치해 줍니다.


4단계: 첫 번째 분석 파일(ipynb) 만들기
데이터 분석가들은 .py 파일보다 .ipynb (Jupyter Notebook) 파일을 훨씬 선호합니다. 코드 실행 결과를 바로 밑에서 볼 수 있거든요.
1. Ctrl + N을 눌러 새 파일을 만든 뒤, 저장(Ctrl + S)할 때 파일명을 **test.ipynb**로 저장하세요.
1. 오른쪽 상단에 **[Select Kernel]**이라는 버튼이 뜨면, 방금 설치한 파이썬 버전을 선택해 주세요.
